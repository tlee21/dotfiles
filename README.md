
## VS Code key settings


Copy ```keybindings.json``` to your vscode settings dir.
```
~/.config/Code/User/
```
### New Keys
- ```c-k 1``` Join all editor groups
- ```c-k 2``` Split Editor Orthogonal
- ```c-k 3``` Split Editor

### Relevant keys
- ```c-1``` Switch to editor group 1
- ```c-2``` Switch to editor group 2
- ```c-3``` etc.
